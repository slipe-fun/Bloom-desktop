# Bloom Desktop

Bloom Desktop is a modern messenger built with **Tauri**, **React**, and **TypeScript**.
It offers post-quantum encryption, open-source code, a simple API, and more to come.

We use **Tailwind CSS** for styling—please keep all UI consistent with it.

---

## Features

- Post-quantum encryption for secure messaging
- Open-source and transparent code
- Simple API for integrations
- Built with modern web technologies and Tauri

---

## Build & Run

1. Install dependencies:

```bash
npm install
```

2. Run in development mode:

```bash
npx tauri dev
```

3. Build for production:

```bush
npx tauri build
```

## Pull Request Guidelines

- Always use Tailwind classes for styling; avoid custom CSS unless necessary
- Write clean, readable TypeScript code
- Include a brief description of your changes in the PR
- Ensure the app builds and runs before submitting a PR
- Keep commits small and focused

## Contribution Tips

- Use existing components and styles where possible
- Test your changes in both development and production builds
